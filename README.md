# ⏱️ Modern Stopwatch Web Application

A responsive stopwatch application developed using HTML, CSS, and JavaScript. It features a modern glassmorphism interface with start, pause, reset, and lap recording functionality.
🚀 Features includes - Start Time- Pause Timer- Reset Timer- Lap Time Recording- Glassmorphism UI- Responsive Design- Attractive Gradient Background.
🛠️ Tech Stack include - HTML5- CSS3- JavaScript
